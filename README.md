## Getting started
### Install
- Open git bash
- Clone the repository : `git clone https://github.com/christian-aucane/auto-sokoban.git`
- Move in the repo : `cd lingua-franca`
- Launch install.sh : `source scripts/install.sh`
### Use
- Open git bash
- Launch run.sh : `source scripts/run.sh`
